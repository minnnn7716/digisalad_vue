# digisalad_vue
### [DEMO 頁面](https://minnnn7716.github.io/digisalad_vue/)

## 儲存庫分支
* main - 主要原始程式碼
* gh-pasge - 編譯建構後程式碼

## 專案指令
### 安裝相關專案
```sh
npm install
```
### 運行環境

```sh
npm run dev
```
### 建構專案
```sh
npm run build
```

## 專案套件
* Node v20.11.0
* Vue v3.4.15
* Vite v5.0.12
* Bottstrap v5.1.3
* Swiper v11.0.5
* AOS v2.3.4
